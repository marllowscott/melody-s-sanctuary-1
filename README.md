# Melody's Digital Sanctuary

A professional website for Melody, a communication coach specializing in helping executives, entrepreneurs, and leaders master the art of powerful communication.

## Features

- Responsive hero section with compelling call-to-actions
- About section showcasing Melody's expertise
- Services overview including 1:1 coaching, executive presence, and public speaking
- Client testimonials
- Newsletter signup for communication tips
- Contact and booking information

## Technologies Used

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- React Router for navigation

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Deployment

This project can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.
